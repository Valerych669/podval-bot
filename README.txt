ИНСТРУКЦИЯ ПО ЗАПУСКУ НА RAILWAY

1. Перейдите на https://railway.app и зарегистрируйтесь.
2. Создайте новый проект: New Project → Empty Project.
3. Зайдите внутрь проекта и нажмите "Add New" → "Empty Service".
4. Загрузите следующие файлы:
   - bot.py
   - requirements.txt
   - Procfile

5. Перейдите во вкладку Variables (переменные окружения).
